# TodoList V3

A modern task management web application built with HTML, CSS, and JavaScript, with companion Swift UI implementation.

## Features

- Task creation and management
- Task details and editing
- User authentication
- Notifications system
- Clean and modern UI design
- Responsive layout

## Project Structure

- HTML/CSS/JS web application
- Swift UI native application for iOS/macOS
- SVG assets for UI elements

## Getting Started

### Web Version
1. Clone this repository
2. Open `index.html` in your browser

### iOS/macOS Version
1. Open the Xcode project file
2. Build and run in simulator or device
